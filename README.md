## Installations

1. clone this repo
```bash
git clone https://github.com/akrindev/neovim-config.git
```
2. copy all folders to your $HOME directory
```bash
cp -r neovim-config/* $HOME
```
3. install plug 
```bash
:PlugInstall
```

4. auth the codeium
```bash
:Codeium Auth
```
